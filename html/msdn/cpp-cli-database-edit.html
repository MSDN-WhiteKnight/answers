C++/CLI - Редактирование базы данных в Windows Forms

Date: 27.02.2015 7:00:15

>чтоб можно было редактировать базу в форме Windows Forms C++

Редактировать содержимое таблицы, имеете в виду? Создайте DataSet, перетащите на него таблицу из обозревателя серверов - он создаст таблицу и класс TableAdapter. На форме создайте DataGridView, укажите источником данных таблицу в DataSet, вызовите методы Fill / Update класса TableAdapter когда нужно загрузить/сохранить данные.
