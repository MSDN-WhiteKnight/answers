SQL Server management studio умеет работаться только с БД SQL Server. Oracle SQL Developer, если верить описаниям, умеет работать с БД и Oracle и SQL Server (хотя второе, возможно, с ограниченным функционалом). Integration Services умеет работать с разными источниками данных, скорее всего, с любым, для которого есть ODBC-драйвер, независимо от наличия обоих предыдущих продуктов. Visual Studio нужен для работы с SSIS и SSAS в графическом режиме, опять же, независимо от наличия предыдущих продуктов.

Т.е., Oracle SQL Developer вам понадобится, если нужно работать именно с движком СУБД Oracle, и только для создания БД, не для работы с SSIS/SSAS. Если есть готовая БД или скрипт для ее создания, то и он не нужен.
