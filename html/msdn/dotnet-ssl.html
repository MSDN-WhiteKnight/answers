Соединение по SSL в .NET

Date: 21.03.2017 7:48:35

Работа через SSL с точки зрения приложения не сильно отличается от работы без него. Вместо обычного Stream используется SslStream.

При работе через TcpClient это будет выглядеть так:

Stream stream = tcpClient.GetStream ();

SslStream ssl = new SslStream (stream);
ssl.AuthenticateAsClient (Host, null, SslProtocols.Tls12, true);

streamWriter = new StreamWriter (ssl as Stream);

//отправить данные используя StreamWriter...

streamReader = new StreamReader (ssl as Stream);

//получить ответ используя StreamReader...
