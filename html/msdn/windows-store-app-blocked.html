"Возможно под "веб-приложениями" подразумеваются приложения под Windows. Например есть сайт  facebook.com и есть приложение для Windows Facebook. Естественно , что опубликовать его может только сам Facebook. "

Приложение Microsoft Новости попадает в эту же категорию, это обертка для сайта MSN. Проблема в том, что часть рекламы, размещенной на стороннем сайте, оно открывает прямо в окне приложения. Лилия утверждает, что за то же самое ее приложение удалили из магазина, т.е. по правилам реклама должна открываться в новом окне браузера. Как по мне, довольно странное правило, но с другой стороны следовать ему было бы не так уж и трудно.

"Веб приложение — клиент-серверное приложение, в котором клиент взаимодействует с веб-сервером при помощи браузера."

С этим я не спорю. Я лишь пытаюсь сказать, что Microsoft Новости является, также как и клиент Facebook, Вконтакте, является оберткой для сайта. Оно просто открывает WebView с содержимым сайта a.msn.com и добавляет вокруг несколько панелей для удобства навигации. То есть под категорию "Products submitted as web apps" оно вроде подходит, хоть технически это и приложение для Windows
