Параметр repositoryPath в NuGet

Date: 19.11.2018 8:51:23

В файл %appdata%\NuGet\NuGet.Config

добавьте параметр repositoryPath:

<configuration>
...

   <config>
      <add key="repositoryPath" value="c:\packages\" />
   </config>

...
</configuration>
