Windows Forms - Почему WebBrowser не открывает сайт?

Date: 23.05.2017 18:24:16

Не все так просто. Существует (бессмысленная) функция "эмуляции браузера". По умолчанию, используется режим IE 7, даже если установлена более новая версия. Естественно, большинство современных сайтов он даже не откроет. На практике, приходится переключаться на более новую версию. См. Browser emulation
