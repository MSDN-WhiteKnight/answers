Взаимодействие с планировщиком задач в .NET

Date: 02.06.2019 8:01:59

Используя библиотеку COM-объектов Task Scheduler. Подключить ее в .NET можно либо добавив ссылку на TLB  (COM ->  Task scheduler ... type library), либо через nuget-пакет https://www.nuget.org/packages/TaskScheduler/. Пример можно посмотреть здесь: https://ru.stackoverflow.com/a/871220/240512 Работать с удаленными компьютерами по сети можно, если на обоих включен DCOM и сделаны нужные настройки в брендмауере.
