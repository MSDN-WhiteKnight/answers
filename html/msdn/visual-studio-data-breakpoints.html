Отследить изменение переменной в Visual Studio

Date: 22.03.2021 11:58:13

Для начала можно попробовать в контекстном меню "Найти все ссылки" и посмотреть, где переменная присваивается. Переменная в данном случае это что, локальная переменная? В С++ есть возможность поставить точку останова на изменение данных, но в C# это появилось только недавно, и видимо только для свойств: https://devblogs.microsoft.com/visualstudio/break-when-value-changes-data-breakpoints-for-net-core-in-visual-studio-2019/
