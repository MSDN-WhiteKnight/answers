Клонирование объектов в .NET

Date: 26.11.2016 18:00:34

Понятие "дамп памяти" обычно используют для определения всего пространства памяти процесса или ядра ОС. К механике MembeerwiseClone() оно не относится.

Метод MemberwiseClone побитно копирует поля объекта, являющиеся структурами (такие как Int32, Point, Rectangle). Для ссылочных типов - копируется ссылка. Конструкторы действительно не вызываются. Не знаю что значит "перезаписывает адреса". Скорее всего, это бред. Информация о механике MembeerwiseClone доступна на странице документации https://msdn.microsoft.com/ru-ru/library/system.object.memberwiseclone(v=vs.110).aspx
