Использование темной темы Windows 10 в приложениях WinAPI

Date: 25.04.2022 17:33:25

Просто включить думаю никак, эта темная тема работает только для приложений UWP. Можно получить текущие настройки темы через API UWP или через реестр, и применить вручную к своей программе. Либо использовать недокументированные функции из библиотеки uxtheme.dll. В интернете гуляют рецепты на эти оба способа: https://stackoverflow.com/q/53501268/8674428, https://stackoverflow.com/q/51334674/8674428

Если вы пишете под Windows 10-11, может перейти на UWP/WinUI? Хоть .NET 6 и самый новый, Windows Forms не получала серьезных обновлений с VS2010.  Сейчас в планах у разработчиков только добавить нормальный API для получения текущей темы: https://github.com/dotnet/winforms/issues/6270, то есть чтобы первый из моих способов можно было реализовать без хаков. Полноценную поддержку темной темы неизвестно когда даже начнут делать.
