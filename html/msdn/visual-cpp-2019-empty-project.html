Создать пустой проект в Visual C++ 2019

Date: 13.01.2021 11:20:43

Если это С++, то систему проектов переделали примерно начиная с VS2017. Теперь нет мастера создания общего Win32-проекта, а вместо него есть отдельные типы проектов: библиотека, консольное приложение, GUI приложение и т.п. Галочки "Пустой проект", соответственно, теперь нет. Но она не очень то и нужна, ибо можно просто вручную удалить все файлы из проекта (+ отключить предварительно компилируемые заголовки). Это даст в итоге тот же результат.
