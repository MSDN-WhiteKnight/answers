Этот вопрос довольно расплывчатый, но, если вы спрашиваете про то, является ли ODBC устаревшим, ответ "Нет". Это вполне поддерживаемый API и именно он считается предпочтительным для взаимодействия с SQL Server из неуправляемого кода. 

https://docs.microsoft.com/en-us/sql/connect/odbc/microsoft-odbc-driver-for-sql-server?view=sql-server-ver15

ODBC is the primary native data access API for applications written in C and C++ for SQL Server. There is an ODBC driver for most data sources. Other languages that can use ODBC include COBOL, Perl, PHP, and Python. ODBC is widely used in data integration scenarios.
