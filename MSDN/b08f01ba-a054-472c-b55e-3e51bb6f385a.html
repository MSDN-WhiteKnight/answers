1. rtb должен быть public

2. Используйте событие BackgroundWorker.ProgressChanged и метод ReportProgress для отображения хода операции на интерфейсе.

Если нужно именно в DoWork, то как-то так:

public delegate void MyDelegate();

public void DoWork()
{   
   secondView.Dispatcher.BeginInvoke(new MyDelegate(DelegateMethod), null);
}

public void DelegateMethod()
{
   secondView.rtb.Text="hello";
}
