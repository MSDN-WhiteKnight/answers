Прикол в том, что таблица символов похоже не переваривает коды больше FFFF, поэтому U+1F844 вы не увидите, даже если он есть в шрифте. В Courier New его конечно же нету. В WPF для диапазона 1F800-1F8FF в файле C:\Windows\Microsoft.NET\Framework\...\WPF\Fonts\GlobalUserInterface.CompositeFont заданы такие привязки:

<!-- Segoe UI Symbol covers all on Win8; Cambria Math covers 2900-2AFF; MS Gothic has little coverage -->
<!-- Segoe UI Emoji supports only emoji characters in these ranges; not adding to avoid uneven display -->
<FontFamilyMap
                Unicode     = "27C0-2BFF, 1F800-1F8FF"
                Target      = "Segoe UI Symbol, Cambria Math, MS Gothic"
                Scale       = "1.0" />
Так что стрелка в одном из этих трех шрифтов.
