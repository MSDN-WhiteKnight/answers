Отправка ATA-команд устройству в Windows

Date: 24.10.2017 8:54:34

Посмотрите в документации по АТА-коммандам. Нужно в ioctl передать структуру ATA_PASS_THROUGH_EX с установленным кодом команды и смещением буфера, и потом буфер данных определенной структуры. Только аккуратно, начните с более простых команд.
