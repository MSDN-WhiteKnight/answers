WinAPI - Запуск процесса с перенаправлением вывода

Date: 13.06.2019 10:29:23

Используя функцию CreateProcess. Пример запуска процесса с перенаправлением вывода см. здесь: https://docs.microsoft.com/en-us/windows/desktop/ProcThread/creating-a-child-process-with-redirected-input-and-output (естественно, ни о каком запуске в одну строку речь не идет, для этого нужен очень громоздкий код). Возможно, проще будет использовать не WinAPI, а синтаксис cmd.exe для перенаправления вывода в файл:

cmd.exe /c python ... > output.txt


После этого считать результат из файла output.txt
