FYI, lib and dll are completely different story. Dll is updated with windows update, but lib file is a part of Windows SDK. If some function was introduced in recent windows version, it must appear in DLL (and thus become usable via GetProcAddress) immediately after OS update. But in order to use it via lib-style-linking you must update Windows SDK as well.

Для информации, lib и dll - это абсолютно разные вещи. DLL обновляется вместе с Windows, но lib-файл является частью Windows SDK. Если какая-то функция появилась только в недавней версии Windows, она немедленно появится в DLL после обновления ОС (так что ее сразу можно будет использовать через GetProcAddress). Но для того, чтобы воспользоваться связыванием через lib-файл, понадобится также обновить Windows SDK.
