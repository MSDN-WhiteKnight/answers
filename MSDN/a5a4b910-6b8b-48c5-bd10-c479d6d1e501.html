"Можно ли скомпилировать проект Visual Studio в другой среде программирования?"

Без изменений, очевидно нельзя. Можно попробовать руками перенести исходный код и настройки проекта. Только зачем? 

"Может можно скомпилировать без среды программирования компилятором как нибудь"

Скомпилировать компилятором, несомненно, можно. В Program Files\Microsoft Visual Studio X.X\VC\bin лежит CL.EXE, ему и скармливаете ваши файлы исходного кода. (См. руководство) Потом результат скармливаете LINK.EXE (там же), и готово.
