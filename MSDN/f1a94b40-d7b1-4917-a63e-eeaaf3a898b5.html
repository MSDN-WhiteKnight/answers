Мнение профи по PowerShell/cmd на TechNet ,я думаю, надо спрашивать. Почему раздел "Безопасность"? Ну так, навскидку - Powershell слегка тяжелее, чем cmd, поэтому если нужно запустить только пару команд, обычно все еще используют cmd. Для сложных скриптов, конечно, никто уже не использует cmd.

Не стоит сильно бояться, что PowerShell где-то будет отключен, так как это все равно сломает слишком многое, и вы не обязаны уметь поддерживать все варианты сломанной конфигурации Windows. Например, даже установка SQL Server 2008 в Windows XP требует PowerShell (которого в стандартной поставке WinXP нету, он ставится с отдельным обновлением).
