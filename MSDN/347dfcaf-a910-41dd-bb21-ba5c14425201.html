Если по простому, то так:

#include <stdio.h>
#include <tchar.h>
#include <windows.h>

int _tmain(int argc, _TCHAR* argv[])
{
	
	while(true)
	{
		printf("Hello, Win32!");
		Sleep(500);
		system("cls");
		Sleep(500);
	}

	return 0;
}
(Параметр 500 - в миллисекундах).

Если нужно при этом еще и принимать ввод пользователя, то отдельный поток делать надо. Какая конкретно задача?
