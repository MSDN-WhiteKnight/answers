"полагаю нужно заключить блок чтения/записи в оператор lock(this){}"

Нет, lock тут не поможет. Для каждого запроса будет свой объект Page в своем домене приложений (или даже процессе, в зависимости от конфигурации IIS).

В данном случае, легкого способа реализовать то что вы хотите на файлах нету. Можно попробовать именованные мьютексы.  Или используйте СУБД (SQL Server Express, MySQL или SQL Server Compact в зависимости от того, что доступно на хостинге), это намного удобнее.
